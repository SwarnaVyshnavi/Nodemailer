Step 1 :Install node in your system.

Step 2 :Install nodemailer module using 'npm i nodemailer' command

Step 4 :In place of user and pass in app.js write sender Email and Password.

step 5 :In options object write the subject and body of Email. Assign receiver mail id in to 'to' attribute in options object.  

step 6 :Run app.js using 'node app' command.

step 7 :Check the inbox or spam for sent mail.
